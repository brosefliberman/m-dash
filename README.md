# m-dash
a local first, ai and science tool dashboard written in rust. includes, an llm interface with a phone link, a basic tts studio, dolly movement, a node graph, basic ml programing, a puzzle feature that is a secret ai/ml powerhouse that can program to do basically anything. optimized for Mac and linux, might open source we will see...
