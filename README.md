# m-dash
A Local First, AI/science tool dashboard written in Rust. Includes: an LLM interface with a phone link, a basic tts studio, dolly movement, a node graph, basic ml programing, a puzzle feature that is a secret ai/ml powerhouse...can mod to do basically anything. Optimized for Mac and linux, might open source we will see...Phone companion app has a semi working beta called jux- link will go here, can link your phone with your locally run llm.
